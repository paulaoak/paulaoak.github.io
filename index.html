Uninformative Priors

MathJax TeX Test Page

Objective or uninformative prior distributions have several advantages, such that they can be used as benchmark that will not reflect the particular biases of the statistician and that they lead to statistical methods which are similar to those developed using classical frequentist theory.

1. Flat Prior

MathJax TeX Test Page We place independent uniform priors on \(\log \sigma_u\) and \(\xi\), which leads to the prior: \begin{equation*} \pi(\xi, \sigma_u)\propto\frac{1}{\sigma_u}. \end{equation*} The main drawback of this choice is that it is not invariant to reparameretization. For example, in our case, \(\sigma_u\) does not follow a uniform distribution.

2. Jeffreys Prior

MathJax TeX Test Page

Jeffreys prior density function is proportional to the square root of the determinant of the Fisher information matrix, that is, \begin{equation*} \pi(\xi, \sigma_u)\propto \sqrt{\det(I(\xi, \sigma_u))}. \end{equation*} The motivation is that it leads to a prior which is invariant to reparameretization.

Jeffreys prior for the iid case and unrounded observations has the following expression: \begin{equation*} \pi(\xi, \sigma_u)\propto \frac{1}{\sigma_u(\xi+1)\sqrt{2\xi+1}}. \end{equation*} We note that, Jeffreys prior leads to distribution where \(\sigma_u\) and \(\xi\) are independent, and where \(\log \sigma_u\) follows a uniform distribution. To ensure propriety of the posterior we need to truncate the distribution imposing the finite lower bound \(\xi>-1/2\).

3. MDI Prior

MathJax TeX Test Page

The MDI prior is defined as \begin{equation*} \pi(\xi, \sigma_u)\propto \exp(\mathbb{E}[\log f(Y_1,\dots,Y_n|\xi, \sigma_u)]), \end{equation*} where \(f(Y_1,\dots,Y_n|\xi, \sigma_u)\) refers to the density or probability mass function. This choice leads to a prior for which the increase in average information, provided by the data via the likelihood function, is maximised. Moreover, like Jeffreys prior it is invariant to reparameretization.

The MDI prior for the iid unrounded observation has the following equation: \begin{equation*} \pi(\xi,\sigma_u)\propto \frac{1}{\sigma_u}\exp(-(\xi+1))\propto\frac{1}{\sigma}\exp(-\xi). \end{equation*} We observe that \(\xi\) and \(\sigma_u\) are independent, \(\xi\) follows a shifted exponential distribution and \(\log \sigma_u\) a uniform distribution. As above, we need the constraint \(\xi>-1\) to ensure propriety of the posterior.

Impact of choosing a variable threshold versus selecting a constant higher threshold







We have analysed the influence of choosing a variable threshold versus selecting a constant higher threshold. To inspect this, we have simulated GPD unrounded data with the shape and scale estimates for the Groningen catalogue (\(\xi = -0.084\), \(\sigma_u = 0.48\)) and variable threshold \(v_i= 1.1\) for the \(250\) first observations and \(v_i = 1.45\) for the \(250\) last ones. We have chosen a low latent threshold \(u = 1\). We then formed another vector of observations only with the observations that exceed \(\max(v)\). We have performed inference using both vector of observations and different objective priors. The joint posterior distribution of \(\xi\), \(\sigma_u\) is shown in the figures below.

We can observe that the posterior mean of the GPD parameters is more close to the true value when using all the observations rather than just the ones exceeding a higher constant threshold.

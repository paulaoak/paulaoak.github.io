Uninformative Priors

MathJax TeX Test Page

Objective or uninformative prior distributions have several advantages, such that they can be used as benchmark that will not reflect the particular biases of the statistician and that they lead to statistical methods which are similar to those developed using classical frequentist theory.

1. Flat Prior

MathJax TeX Test Page We place independent uniform priors on \(\log \sigma_u\) and \(\xi\), which leads to the prior: \begin{equation*} \pi(\xi, \sigma_u)\propto\frac{1}{\sigma_u}. \end{equation*} The main drawback of this choice is that it is not invariant to reparameretization. For example, in our case, \(\sigma_u\) does not follow a uniform distribution.

2. Jeffreys Prior

MathJax TeX Test Page

Jeffreys prior density function is proportional to the square root of the determinant of the Fisher information matrix, that is, \begin{equation*} \pi(\xi, \sigma_u)\propto \sqrt{\det(I(\xi, \sigma_u))}. \end{equation*} The motivation is that it leads to a prior which is invariant to reparameretization.

Jeffreys prior for the iid case and unrounded observations has the following expression: \begin{equation*} \pi(\xi, \sigma_u)\propto \frac{1}{\sigma_u(\xi+1)\sqrt{2\xi+1}}. \end{equation*} We note that, Jeffreys prior leads to distribution where \(\sigma_u\) and \(\xi\) are independent, and where \(\log \sigma_u\) follows a uniform distribution. To ensure propriety of the posterior we need to truncate the distribution imposing the finite lower bound on \(\xi\), \(\xi>-1/2\).

3. MDI Prior

MathJax TeX Test Page

The MDI prior is defined as \begin{equation*} \pi(\xi, \sigma_u)\propto \exp(\mathbb{E}[\log f(Y_1,\dots,Y_n|\xi, \sigma_u)], \end{equation*} where \(f(Y_1,\dots,Y_n|\xi, \sigma_u)\) refers to the density or probability mass function. This choice leads to a prior for which the increase in average information, provided by the data via the likelihood function, is maximised. Moreover, like Jeffreys prior it is invariant to reparameretization.\newline

The MDI prior for the iid unrounded observation has the following equation: \begin{equation*} \pi(\xi,\sigma_u)\propto \frac{1}{\sigma_u}\exp(-(\xi+1))\propto\frac{1}{\sigma}\exp(-\xi). \end{equation*} We observe that \(\xi\) and \(\sigma_u\) are independent, \(\xi\) follows a shifted exponential distribution and \(\log \sigma_u\) a uniform distribution. As above, we need the constraint \(\xi>-1\) to ensure propriety of the posterior.
